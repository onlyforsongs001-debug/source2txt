# Multi-Agent Coordination Protocol

## Quick Start cho Agent mới
1. **Đọc file này trước khi làm gì**
2. **Đăng ký** tên agent ở mục `👥 Agent Registry`
3. **Nhận task** từ Task Board hoặc từ user chat
4. **Lock files** trước khi sửa ở mục `🔒 Active Locks`
5. **Unlock + báo done** khi xong

## 🔒 File Lock System
Mỗi lần sửa file, ghi vào đây. Release ngay khi xong.

| Agent | Files đang giữ | Bắt đầu | Tự động hết hạn |
|---|---|---|---|
| _trống_ | - | - | - |

**Rules:**
- Lock TỐI ĐA 30 phút, nếu quá thì agent khác có quyền force unlock
- Ko lock > 3 files cùng lúc
- Ghi rõ PATH đầy đủ của file

## ⚡ Cách deploy (Vercel + Hugging Face)

### Vercel (UI + API Routes)
```bash
# 1. Push to GitHub
git init && git add . && git commit -m "initial"
gh repo create source2txt --public --push

# 2. Import vào Vercel (vercel.com)
#    - Framework: Next.js
#    - Add all env vars từ .env.local
#    - Deploy
```

### Hugging Face (Backend / Model Hosting)
**Option A - Deploy full app trên HF Space:**
- Tạo Space: huggingface.co → New Space → Docker/Next.js
- Add `Dockerfile` hoặc dùng `huggingface/space-config`

**Option B - Host models riêng trên HF:**
- Upload Whisper/DeepSeek model → HF Inference Endpoint
- API trả về endpoint URL → dùng trong `.env.local`

### ✅ Files deploy đã tạo sẵn:
| File | Mục đích |
|---|---|
| `vercel.json` | Config Vercel (framework, build, env) |
| `Dockerfile` | Cho HF Docker Space (chạy full app) |
| `hf_space.yaml` | HF Space config (model + port) |
| `hf_backend/` | Python FastAPI backend dùng Whisper cho HF |

### 🔧 Kiến trúc đề xuất:
```
User → Vercel (UI + API routes) → HF Backend (transcribe, summarize)
                                       ↓
                              Supabase (DB + Auth)
                                       ↓
                              Stripe (Payments)
```
Hoặc đơn giản: **Vercel chạy full app**, HF chỉ host model API.
